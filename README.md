## How to run

**Web**
```bash
# Enter the web folder with
$ cd NLW_03-Happy/web

# Install the dependencies
$ npm install

# Run the application
$ npm start
```


**Backend**
```bash
# Enter the backend folder with
$ cd NLW_03-Happy/backend

# Install the dependencies
$ npm install

# Run the backend
$ npm start
```


**Mobile**
```bash
# Enter the mobile folder with
$ cd NLW_03-Happy/mobile

# Install the dependencies
$ npm install

# Run the app
$ npm start

# Scan the QR Code displayed in the browser window
```
